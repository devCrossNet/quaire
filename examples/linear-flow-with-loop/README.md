# Example for a linear flow with a loop

The second option of the last question allows the user to go back to the first question.

```
           ┌────────────┐
           │            │
           │ Question 1 │◄──────────────┐
           │            │               │
           └─────┬──────┘               │
                 │                      │
                 │                      │
┌──────────┐     │      ┌──────────┐    │
│ Option 1 ├─────┴──────┤ Option 2 │    │
└────┬─────┘            └─────┬────┘    │
     │                        │         │
     │                        │         │
     │                        │         │
     └────►┌────────────┐◄────┘         │
           │            │               │
           │ Question 2 │               │
           │            │               │
           └─────┬──────┘               │
                 │                      │
                 │                      │
┌──────────┐     │      ┌──────────┐    │
│ Option 1 ├─────┴──────┤ Option 2 │    │
└────┬─────┘            └─────┬────┘    │
     │                        │         │
     │                        │         │
     │                        │         │
     └────►┌────────────┐◄────┘         │
           │            │               │
           │ Question 3 │               │
           │            │               │
           └─────┬──────┘               │
                 │                      │
                 │                      │
┌──────────┐     │      ┌──────────┐    │
│ Option 1 ├─────┴──────┤ Option 2 ├────┘
└──────────┘            └──────────┘
```
