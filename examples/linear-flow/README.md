# Example for a linear flow

This is a very simple flow with a couple of single select questions.

```
           ┌────────────┐
           │            │
           │ Question 1 │
           │            │
           └─────┬──────┘
                 │
                 │
┌──────────┐     │      ┌──────────┐
│ Option 1 ├─────┴──────┤ Option 2 │
└────┬─────┘            └─────┬────┘
     │                        │
     │                        │
     │                        │
     └────►┌────────────┐◄────┘
           │            │
           │ Question 2 │
           │            │
           └─────┬──────┘
                 │
                 │
┌──────────┐     │      ┌──────────┐
│ Option 1 ├─────┴──────┤ Option 2 │
└────┬─────┘            └─────┬────┘
     │                        │
     │                        │
     │                        │
     └────►┌────────────┐◄────┘
           │            │
           │ Question 3 │
           │            │
           └─────┬──────┘
                 │
                 │
┌──────────┐     │      ┌──────────┐
│ Option 1 ├─────┴──────┤ Option 2 │
└──────────┘            └──────────┘
```
